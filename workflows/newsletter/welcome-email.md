# Workflow: Newsletter Welcome Flow (Virginia Homegrown)

## Objective
Collect subscriber name and interest data via a post-signup welcome email that links to a Mailchimp Update Profile form — no separate landing page required.

## Collaborators
Mike, Michael, Leandra

## Inputs
- New subscriber email (triggered on signup)
- Mailchimp list/audience ID for Virginia Homegrown
- Update Profile form URL (generated in Mailchimp)

## Steps
1. Subscriber signs up for Virginia Homegrown newsletter
2. Mailchimp automation triggers welcome email (1–2 hours post-signup)
3. Welcome email includes:
   - Warm intro to the newsletter
   - Link to Mailchimp Update Profile form asking for name + interests
4. Subscriber fills out form → data saved to their Mailchimp profile
5. Use interest data for future segmentation

## Output
- Subscriber profile enriched with name and interest tags
- Segmented audience available for targeted sends

## Mailchimp Notes
- Update Profile form is built inside Mailchimp (Forms → Update Profile)
- The form link is unique per subscriber — Mailchimp handles the merge tag
- No separate landing page or custom form needed
